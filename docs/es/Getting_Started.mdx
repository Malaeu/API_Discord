# Creando tu primera aplicación de Discord
Las aplicaciones de Discord te permiten personalizar y ampliar tus servidores utilizando una colección de APIs y funciones interactivas. Esta guía te enseñará a crear tu primera aplicación de Discord con JavaScript, y al final tendrás una aplicación que utiliza comandos de barra, envía mensajes y responde a las interacciones de los componentes.

Construiremos una aplicación de Discord que permita a los miembros del servidor jugar piedra, papel o tijera (con 7 opciones en lugar de las 3 habituales). Esta guía está enfocada para principiantes, pero asume que tienes una comprensión básica de [JavaScript](https://developer.mozilla.org/en-US/docs/Learn/Getting_started_with_the_web/JavaScript_basics).

<Collapsible title="Lo que crearemos" icon="view">
  Así será el resultado de la aplicación:

  ![Demo of example app](getting-started-demo.gif)

  Para hacer el flujo del usuario un poco más explícito:
  1. El usuario 1 inicia un nuevo juego y elige su objeto utilizando el comando de barra `/challenge` de la aplicación.
  2. Se envía un mensaje al canal con un botón que invita a otros a aceptar el desafío.
  3. El usuario 2 presiona el botón **Accept**
  4. Se envía un mensaje efímero al usuario 2, donde selecciona su objeto de elección.
  5. El resultado del juego se publica nuevamente en el canal original para que todos lo vean.
